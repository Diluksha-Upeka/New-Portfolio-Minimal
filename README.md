## Getting Started

1. **Install dependencies:**
```bash
npm install
```

2. **Run the development server:**
```bash
npm run dev
```

3. **Open [http://localhost:3000](http://localhost:3000)**

## Configuration

Edit `lib/github.ts` and replace `YOUR_USERNAME` with your GitHub username to automatically fetch your repositories.
